# Watering::Can

This Rails project app is designed for plant owners to keep their plants watered while they are on vacation.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'watering-can'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install watering-can
